# Drone Service Application

A C# WPF desktop application designed using a class library architecture.

---

## 🚀 Overview

This project demonstrates object-oriented programming and separation of concerns by dividing UI and business logic.

---

## ✨ Features

* Service management system
* WPF user interface
* Class library for backend logic

---

## 🛠️ Tech Stack

* C#
* .NET
* WPF

---

## 📂 How to Run

1. Open solution in Visual Studio
2. Build project
3. Run application

## 📚 Learning Outcomes

* OOP concepts
* Desktop application development
* Application architecture
